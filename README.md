# Draw-app
Drawing App using HTML, CSS, and JavaScript
project structure
drawing
├── index.html
├── style.css
└── script.js
A simple browser-based drawing application built using HTML, CSS, and JavaScript.

## Features
- Draw using mouse
- Change brush color
- Adjustable brush size
- Eraser tool
- Clear canvas
- Save drawing as PNG image

## Technologies Used
- HTML5 Canvas
- CSS3
- JavaScript

## How to Run

1. Download or clone the project
2. Open the folder
3. Open `index.html` in your browser
